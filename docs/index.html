ghibli-clouds-new

Unity plugin for procedural, stylized cloud generation.

Included Runtime Files

  • Assets/GhibliClouds/Runtime/WorleyNoise3D.cs
  • Assets/GhibliClouds/Runtime/GhibliCloudMetaMesh.cs

Quick Usage

  1. Add the Assets/GhibliClouds folder to your Unity project.
  2. Create a GameObject with a MeshFilter and MeshRenderer.
  3. Add GhibliClouds.Runtime.GhibliCloudMetaMesh.
  4. Tune noise and metaball parameters as needed.
  5. Use Regenerate Cloud Meta Mesh to rebuild after changes.
